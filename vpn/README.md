# 🛡️ Собственный AmneziaWG VPN Сервер

**AmneziaWG** — это форк популярного протокола WireGuard, разработанный командой Amnezia VPN специально для работы в условиях жесткой интернет-цензуры. Он полностью совместим с WireGuard, но защищен от блокировок по сигнатурам с помощью добавления мусорных пакетов в рукопожатие и изменения структуры заголовков.

В этом разделе описана простая установка AmneziaWG-сервера с веб-панелью управления **amnezia-wg-easy** для удобного создания пользователей и скачивания готовых конфигураций/QR-кодов.

---

## 🚀 Установка сервера

### 1. Подготовка папки
Создайте рабочую директорию на вашем сервере VPS:
```bash
mkdir -p /opt/amnezia-wg && cd /opt/amnezia-wg
```
*(Скопируйте в эту папку [`docker-compose.yml`](./docker-compose.yml) из этого репозитория).*

### 2. Редактирование параметров
Откройте `docker-compose.yml` и настройте переменные:
*   `WG_HOST` — укажите публичный IP-адрес вашего VPS.
*   `PASSWORD` — укажите надежный пароль для входа в веб-интерфейс панели управления.

### 3. Открытие портов
В брандмауэре вашего хостинга откройте порты:
*   `51820 / UDP` — для VPN трафика.
*   `51821 / TCP` — для доступа к панели администратора (рекомендуется закрыть его или ограничить доступ только с вашего IP после настройки, либо использовать SSH-туннель).

### 4. Запуск сервера
Запустите Docker контейнер:
```bash
docker compose up -d
```
Служба автоматически поднимет туннель, настроит NAT-маршрутизацию на сервере и запустит веб-интерфейс.

## 🖥️ Создание пользователей и подключение

1. Откройте браузер и перейдите на страницу: `http://IP-адрес-вашего-сервера:51821`
2. Введите пароль, указанный в `docker-compose.yml`.
3. Нажмите кнопку **+ Client** (Создать клиента) и введите имя пользователя (например, `phone-stepan`).
4. Рядом с созданным пользователем появятся кнопки:
   *   **QR-код** — для быстрого скачивания на телефон.
   *   **Скачать конфиг (.conf)** — для загрузки на ПК или роутер.
   *   **Удалить/Деактивировать** — для управления доступом.

Конфигурационный файл, скачиваемый из этой панели, уже содержит все необходимые настройки обхода DPI (`Jc`, `Jmin`, `Jmax`, `S1`, `S2`, `H1-H4`), поэтому никаких дополнительных ручных правок делать не нужно.

## 📱 Клиенты для AmneziaWG

### 💻 Компьютеры (Windows, macOS, Linux)
1. Скачайте официальный клиент AmneziaWG:
   *   [Релизы на GitHub](https://github.com/amnezia-vpn/amneziawg-windows/releases) (для Windows)
   *   [Сайт Amnezia VPN](https://amnezia.org/ru/amneziawg) (все платформы)
2. Импортируйте скачанный `.conf` файл.
3. Нажмите кнопку **Подключить**.

### 📱 Смартфоны (Android, iOS)
1. Установите приложение **AmneziaWG** из Google Play или App Store.
2. Откройте приложение, нажмите **+** -> **Сканировать QR-код**.
3. Отсканируйте код из веб-интерфейса сервера и подключитесь.

### 🔌 Настройка на роутерах Keenetic
1. Скачайте `.conf` файл клиента из веб-интерфейса сервера.
2. Зайдите в веб-интерфейс Keenetic -> **Другие подключения** -> **Добавить подключение**.
3. Выберите тип: **AmneziaWG**.
4. Заполните поля из `.conf` файла (Приватный ключ, Адрес, DNS, публичный ключ пира, Endpoint и параметры обхода DPI: `Jc=4`, `Jmin=50`, `Jmax=1000`, `S1=15`, `S2=44`, `H1=1`, `H2=2`, `H3=3`, `H4=4`).
5. Сохраните подключение и назначьте его основным или настройте маршруты для конкретных домашних устройств.
